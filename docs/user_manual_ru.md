# AssetHub - Руководство пользователя

## Введение

AssetHub - это универсальная платформа для управления ресурсами в 3ds Max, которая помогает дизайнерам эффективно организовывать, искать и использовать 3D-модели, текстуры и материалы. Платформа предоставляет централизованный доступ к локальным и облачным библиотекам ресурсов, значительно упрощая рабочий процесс визуализации проектов.

## Основные возможности

- **Каталогизация ресурсов**: Автоматическое сканирование и индексация локальных 3D-моделей, текстур и материалов
- **Интеллектуальный поиск**: Быстрый поиск ресурсов по имени, категории, типу и другим параметрам
- **Интеграция с внешними библиотеками**: Доступ к популярным онлайн-библиотекам 3D-моделей (Turbosquid, CGTrader)
- **Плагин для 3ds Max**: Прямая интеграция с 3ds Max для импорта моделей и текстур
- **Удобный интерфейс**: Интуитивно понятный пользовательский интерфейс для эффективной работы с ресурсами

## Установка

### Системные требования

- Windows 10 или новее
- Python 3.9-3.11
- 3ds Max 2020 или новее (для функций плагина)
- 4 ГБ оперативной памяти (рекомендуется 8 ГБ)
- 500 МБ свободного места на диске (не включая библиотеку ресурсов)

### Установка из исходного кода

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/assethub.git
   cd assethub
   ```

2. Установите зависимости:
   ```bash
   pip install -e .
   ```

3. Запустите приложение:
   ```bash
   python main.py
   ```

### Установка плагина для 3ds Max

1. Скопируйте файлы плагина в директорию скриптов 3ds Max:
   ```
   %APPDATA%\Autodesk\3ds Max 20XX\ENU\scripts\startup\
   ```

2. Перезапустите 3ds Max

3. Запустите плагин через меню MAXScript:
   ```
   MAXScript > Run Script > выберите max_integration.py
   ```

## Начало работы

### Первый запуск

При первом запуске AssetHub создаст необходимые директории и конфигурационные файлы:

- `~/.assethub/config.json` - конфигурационный файл
- `~/.assethub/database.db` - база данных ресурсов
- `~/.assethub/assets` - директория для локальных ресурсов
- `~/.assethub/index` - поисковый индекс

### Сканирование ресурсов

1. Запустите AssetHub
2. Нажмите кнопку "Scan Directory" в боковой панели
3. Выберите директорию с вашими 3D-моделями, текстурами и материалами
4. Дождитесь завершения сканирования
5. Подтвердите индексацию найденных ресурсов

### Поиск ресурсов

1. Введите поисковый запрос в поле поиска
2. При необходимости используйте фильтры по категории и типу файла
3. Нажмите кнопку "Search" или клавишу Enter
4. Просмотрите результаты поиска в списке

### Импорт ресурсов в 3ds Max

#### Через плагин

1. Запустите плагин AssetHub в 3ds Max
2. Найдите нужный ресурс через поиск
3. Дважды щелкните на ресурсе или выберите его и нажмите "Import Selected"

#### Через основное приложение

1. Найдите нужный ресурс через поиск
2. Дважды щелкните на ресурсе для просмотра деталей
3. Скопируйте путь к файлу
4. Импортируйте файл в 3ds Max стандартными средствами

### Поиск в онлайн-библиотеках

1. Перейдите на вкладку "Online Assets"
2. Выберите провайдера (Turbosquid или CGTrader)
3. Введите поисковый запрос
4. Нажмите кнопку "Search Online"
5. Просмотрите результаты поиска

## Основные функции

### Каталогизация ресурсов

AssetHub автоматически сканирует указанные директории и индексирует найденные ресурсы. Поддерживаются следующие типы файлов:

- **Модели**: .obj, .fbx, .3ds, .max, .blend, .dae, .stl, .ply
- **Текстуры**: .jpg, .jpeg, .png, .tif, .tiff, .bmp, .tga, .hdr, .exr
- **Материалы**: .mtl, .mat, .xml, .json

При сканировании AssetHub извлекает метаданные из файлов, такие как размеры, количество полигонов и вершин, и сохраняет их в базе данных для быстрого поиска.

### Поиск ресурсов

AssetHub предоставляет мощный поисковый движок для быстрого поиска ресурсов. Вы можете искать по:

- Имени файла
- Описанию
- Категории
- Тегам
- Типу файла
- Формату файла

Результаты поиска отображаются в виде списка с предпросмотром, что позволяет быстро найти нужный ресурс.

### Интеграция с внешними библиотеками

AssetHub интегрируется с популярными онлайн-библиотеками 3D-моделей:

- **Turbosquid**: Огромная библиотека 3D-моделей, текстур и материалов
- **CGTrader**: Маркетплейс для 3D-моделей и дизайнеров

Для использования этих библиотек вам потребуется API-ключ, который можно получить на соответствующих сайтах.

### Плагин для 3ds Max

Плагин AssetHub для 3ds Max предоставляет прямой доступ к вашей библиотеке ресурсов прямо из 3ds Max. Вы можете:

- Искать ресурсы в локальной библиотеке
- Импортировать модели и текстуры в текущую сцену
- Искать ресурсы в онлайн-библиотеках
- Скачивать и импортировать ресурсы из онлайн-библиотек

## Советы и рекомендации

### Организация библиотеки

Для эффективной работы с AssetHub рекомендуется организовать вашу библиотеку ресурсов следующим образом:

```
assets/
├── models/
│   ├── furniture/
│   ├── architecture/
│   └── characters/
├── textures/
│   ├── wood/
│   ├── metal/
│   └── fabric/
└── materials/
    ├── wood/
    ├── metal/
    └── glass/
```

Такая структура позволит AssetHub более эффективно категоризировать ваши ресурсы.

### Оптимизация поиска

- Используйте конкретные поисковые запросы для более точных результатов
- Применяйте фильтры по категории и типу файла для сужения результатов
- Регулярно обновляйте индекс после добавления новых ресурсов

### Работа с онлайн-библиотеками

- Сохраняйте часто используемые ресурсы локально для быстрого доступа
- Используйте теги и категории для организации скачанных ресурсов
- Проверяйте лицензии на использование скачанных ресурсов

## Устранение неполадок

### Приложение не запускается

- Убедитесь, что установлены все зависимости
- Проверьте версию Python (требуется 3.9-3.11)
- Проверьте права доступа к директории ~/.assethub

### Ошибки при сканировании

- Убедитесь, что у вас есть права на чтение сканируемой директории
- Проверьте, что файлы имеют поддерживаемые форматы
- Убедитесь, что файлы не повреждены

### Плагин не работает в 3ds Max

- Убедитесь, что файлы плагина находятся в правильной директории
- Проверьте версию 3ds Max (требуется 2020 или новее)
- Проверьте журнал ошибок 3ds Max на наличие сообщений об ошибках

### Проблемы с поиском

- Перестройте индекс через кнопку "Rebuild Index"
- Убедитесь, что ресурсы были успешно проиндексированы
- Проверьте правильность поискового запроса

## Заключение

AssetHub - это мощный инструмент для управления ресурсами, который значительно упрощает работу дизайнеров, создающих визуализации проектов в 3ds Max. Благодаря автоматической каталогизации, интеллектуальному поиску и интеграции с внешними библиотеками, AssetHub позволяет сосредоточиться на творческих аспектах работы, а не на поиске и организации ресурсов.

Мы надеемся, что AssetHub станет незаменимым инструментом в вашем рабочем процессе и поможет вам создавать потрясающие визуализации быстрее и эффективнее.
